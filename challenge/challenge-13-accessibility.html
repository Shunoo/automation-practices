Properly Accessible Form

Issues to Find

  • 🔍 Low contrast text (WCAG AA 4.5:1 ratio)
  • 🔍 Form inputs without associated labels
  • 🔍 Non-semantic interactive elements (div as button)
  • 🔍 Images without alt text
  • 🔍 Table without proper header structure
  • 🔍 Modal without focus trap or ARIA attributes
  • 🔍 Color-only status indicators (no text alternative)
  • 🔍 Missing skip navigation link
  • 🔍 Missing landmark roles (nav, main, etc.)
  • 🔍 Keyboard navigation issues
